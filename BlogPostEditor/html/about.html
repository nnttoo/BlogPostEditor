Blog Post Editor Adalah software opensource untuk membuat, update dan edit posting blog tanpa perlu login melalui web.
Dengan Blog Post Editor anda bisa membuat dan menyimpan artikel dalam kondisi offline, artikel akan di simpan dalam database di penyimpanan lokal komputer anda dan kemudian bisa anda publish kapanpun ketika anda sudah terhubung dengan internet.
